#  Autonomous Fire Detection and Suppression Rover

> An intelligent embedded robotics system that autonomously detects fire, aligns toward the flame source, suppresses it using a water pump, avoids obstacles, and supports real-time monitoring through a React-based dashboard.

---

##  Overview

Fire accidents pose significant risks to human life, especially to firefighters operating in hazardous environments. This project presents an autonomous firefighting rover capable of detecting fire, aligning itself toward the flame, activating a water pump for suppression, avoiding obstacles during movement, and providing real-time monitoring through a software dashboard.

The objective is to reduce human intervention during the initial stages of fire emergencies while demonstrating the integration of embedded systems, robotics, and software into a practical safety solution.
